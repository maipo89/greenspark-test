How to Run Locally



Prerequisites

Node.js (v14+)
Yarn (latest version)



Install dependencies:

yarn



Start the app:

yarn dev



Open the app in your browser at:

http://localhost:3000




API

The app fetches widget data from this mock API:

https://b795b019-1f84-41f4-93a3-a702d686c75a.mock.pstmn.io/product-widgets




How to Use



Customize the widget color and toggle activation.

Link/unlink widgets to a public profile.



Features



Customize widget color and activation status.

Link widgets to a public profile.

Fetches data from a mock API.



The app uses Vue.js for dynamic widget management.
SCSS modules are used for styling.
